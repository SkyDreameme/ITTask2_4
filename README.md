# Задание №4 — Иерархия классов файловой системы
Необходимо реализовать иерархию классов для моделирования файловой системы, состоящую из базового класса и двух классов-наследников. Приложение демонстрирует работу с файлами и папками через оконный интерфейс.

Базовый класс FileSystemElement:
Абстрактный класс, представляющий любой элемент файловой системы.

**Производные классы:**

*FileElement (Файл)* \
-Размер задаётся при создании в конструкторе \
-Не имеет дочерних элементов \
-Тип: ElementType.File 

*FolderElement (Папка)* \
-Размер вычисляется как сумма размеров всех вложенных элементов \
-Содержит коллекцию дочерних элементов \
-Тип: ElementType.Folder

**Что используется в проекте:** \
Язык программирования - C# \
Платформа - .NET 10.0 \
UI фреймворк - Avalonia UI \
MVVM библиотека - CommunityToolkit.Mvvm

**Пошаговая инструкция** \
*Копировать элемент* \
1. Выберите файл или папку в дереве
2. Нажмите кнопку «Копировать»
3. Элемент скопируется в ту же папку с новым именем (file1.txt, file2.txt) \
*Переместить элемент* \
1. Выберите файл или папку в дереве
2. Нажмите кнопку «Переместить»
3. В диалоге выберите целевую папку
4. Нажмите OK
5. Элемент переместится в выбранную папку \
*Удалить элемент* \
1. Выберите файл или папку в дереве
2. Нажмите кнопку «Удалить»
3. Элемент будет удалён из файловой системы \
*Просмотр свойств* \
1. Выберите любой элемент в дереве
2. В нижней панели отобразятся: имя элемента, полный путь, размер в байтах, тип (File/Folder)
